FPC AutoSMM

Автоматизация, которая работает на вас

Описание

Плагин на автонакрутку для FunPay Cardinal. Этот плагин создан для автоматизации процессов на платформе FunPay.

Настройка

  • Введите /autosmm в боте после установки плагина.
  • Вставьте ссылку на API сайта в поле API URL, например: https://site.com/api/v2/.
  • Сгенерируйте ключ API и вставьте его в поле API KEY, например: kwjehfefhbvwieoirepdklen354nfdlfk9fn4rf.

Настройка лотов

Лоты можно выставлять в любой категории. В описании лота укажите параметры:

  • ID: (число)
  • #Quan: (число от 2 до ∞) — необязательно

Пример: ID: 365, #Quan: 10.

Требования

Для установки плагина требуется Python 3.11.
